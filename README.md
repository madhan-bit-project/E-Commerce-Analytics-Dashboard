# E-Commerce Analytics Dashboard

## Project Overview

This project is an end-to-end E-Commerce Analytics solution built using Python, SQL, SQLite, and Power BI. The project analyzes customer behavior, sales performance, delivery efficiency, payment methods, and customer reviews to generate actionable business insights.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL
* SQLite
* Power BI

## Project Workflow

1. Data Collection and Preparation
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL-Based Business Analysis
5. KPI Generation
6. Power BI Dashboard Development
7. Business Insights and Reporting

## Key Business Insights

* Analyzed over 99,000 customer orders.
* Evaluated revenue trends across multiple months.
* Identified top-performing customer states.
* Analyzed delivery performance and delivery times.
* Studied customer review patterns and satisfaction levels.
* Examined payment method preferences.
* Identified top-selling product categories.
